# 🛒 Grocery Application

A simple and user-friendly grocery shopping web application.

## 🚀 Features
- Browse grocery products
- Add/remove items from cart
- View total price dynamically
- Responsive and clean UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Description
This project simulates an online grocery shopping experience where users can explore products, manage their cart, and perform basic shopping operations. It demonstrates core frontend development concepts and e-commerce workflow.
